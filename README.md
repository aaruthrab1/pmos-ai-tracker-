# Cyra

**AI-powered women's health companion** — PCOS awareness, symptom tracking, cycle insights, education, and doctor preparation.

## Overview

Cyra helps women understand and manage menstrual and hormonal health through intelligent symptom tracking, personalized educational resources, AI-powered guidance, and doctor visit preparation tools.

## Problem Statement

Millions of women struggle to identify patterns in menstrual health, hormonal changes, and symptoms related to PCOS due to fragmented tracking tools and limited access to personalized guidance. Important symptoms often go undocumented, making it difficult to recognize trends and seek timely medical support.

Cyra addresses this challenge by combining health tracking, educational resources, AI assistance, and health report generation into a single platform.

## Key Features

### 🤖 AI Sakhi
An AI-powered wellness companion that helps users understand menstrual health, cycle tracking, PCOS awareness, nutrition, and lifestyle habits.

### 📊 Period & Symptom Tracking
Track:
- Period dates
- Symptoms
- Mood changes
- Sleep quality
- Wellness indicators

### 📈 Cycle Insights
Visualize patterns and trends using interactive charts and health analytics.

### 📚 Health Journey
Educational content covering:
- Menstrual Health
- PCOS Awareness
- Nutrition
- Exercise
- Stress Management
- Sleep Health

### 📄 Smart Reports
Generate structured health summaries to assist users during healthcare consultations.

### 🌐 Multilingual Support
Accessible health information across multiple languages for wider inclusivity.

### 📱 Progressive Web App
Installable on mobile devices with offline support for core functionality.

## Quick Start

```bash
# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Configure Supabase and Groq credentials

# Run database migrations
supabase db push

# Start development servers
npm run dev
```
